How to Install Your Signature

1. Host the logo. Upload the PNG to your website (e.g., ascendrabio.com/img/email-logo.png). Replace the embedded image with that URL in the HTML.
2. Copy the signature. Open this file in Chrome, highlight just the signature you want inside the white box, and copy it (Ctrl+C / Cmd+C).
3. Gmail: Settings → See all settings → General → Signature → paste.
4. Outlook: Settings → Mail → Compose and reply → Email signature → paste.
5. Apple Mail: Preferences → Signatures → New → paste.
6. Personalize. Replace "John Smith," title, phone, and email with each rep's info. Everything else stays the same across the team.
